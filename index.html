Twitter4Rv0.7.0

Idiomatic Ruby Open Source Library for the Twitter REST & Search APIs

Follow Twitter4R on Twitter => http://twitter.com/t4ruby

Twitter4R News

Project Info

Twitter4R v0.7.0 now provides good coverage of the current Twitter REST & Search API coverage for Ruby clients. Additional features include OAuth, SSL, proxy and user agent ("source") support & customization.
Twitter4R is also customizable to use against identi.ca.
The Twitter4R project is released under the MIT License as a Ruby Gem.

Downloads

Visit the GitHub project repository to download the latest Twitter4R release. All versions are available as a Ruby Gem:

sudo gem install twitter4r
